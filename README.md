# Lista de Tareas - Mantenimiento de Software

Proyecto React básico para la actividad de control de versiones en GitHub.

## Descripción

Aplicación sencilla de gestión de tareas que permite:
- Agregar nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas
- Establecer prioridades (baja, media, alta)

## Cómo ejecutar el proyecto

1. Clona el repositorio
2. Instala las dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`
4. Abre http://localhost:5173 en tu navegador

## Tecnologías utilizadas
- React 18
- TypeScript
- Vite
- CSS puro
