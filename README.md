# Historias de Usuario - Lista de Tareas

1. **Como usuario del sistema, quiero poder filtrar mis tareas por nivel de prioridad (alta, media, baja), para enfocarme en las más importantes primero.**
   - Criterios de aceptación:
     - Botones para filtrar por cada nivel de prioridad
     - Opción para mostrar todas las tareas
     - Indicador visual del filtro activo

2. **Como usuario frecuente, quiero poder editar las tareas existentes, para corregir información o actualizar detalles.**
   - Criterios de aceptación:
     - Botón de edición en cada tarea
     - Formulario de edición con campos prellenados
     - Posibilidad de guardar o cancelar cambios

3. **Como usuario, quiero que mis datos se guarden automáticamente en el localStorage, para no perder mis tareas al cerrar el navegador.**
   - Criterios de aceptación:
     - Persistencia de datos entre sesiones
     - Carga automática de tareas guardadas al iniciar la aplicación
     - Actualización instantánea del localStorage con cada cambio

4. **Como administrador de clínica, quiero ver un panel de resumen con estadísticas de mis tareas, para tener una visión general de mi productividad.**
   - Criterios de aceptación:
     - Contador de tareas totales
     - Porcentaje de tareas completadas
     - Distribución por niveles de prioridad
     - Gráfico visual simple de las estadísticas

5. **Como usuario, quiero poder ordenar mis tareas por fecha de creación o por prioridad, para organizar mi lista según mis preferencias.**
   - Criterios de aceptación:
     - Selector para elegir criterio de ordenamiento
     - Opciones: "Más recientes primero", "Más antiguas primero", "Alta prioridad primero"
     - Actualización inmediata del listado al cambiar el criterioZ
